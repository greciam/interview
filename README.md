Outline
This project uses the following tech stack:
- FastApi to create Apis with documentation
- SqlAlchemy as an ORM to create models
- Pydantic to create API schemas
- Uvicorn as the server
- PgAdmin/Postgresql for database

Steps Required to run Application
- Create a virtual environment
- Create a .env file with database information
- Add .env file to .gitignore

Commands for Running Application (in a development environment)
- cd EhrApp
- fastapi dev main.py
